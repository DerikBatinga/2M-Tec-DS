# Site EtecZL

Website institucional da **Etec Zona Leste**, uma escola técnica vinculada ao Centro Paula Souza (CEETEPS).

## :computer: Tecnologias

- **PHP** — Backend e estruturação das páginas
- **HTML5** — Marcação semântica
- **CSS3** — Estilização (layout, cores, tipografia)
- **JavaScript** — Interatividade

## :file_folder: Estrutura

```
Site-Etec/
├── index.php          # Página inicial
├── index.css          # Estilos da home
├── Cursos/            # Página de cursos técnicos
├── Gestão/            # Página sobre a gestão escolar
├── Contato/           # Formulário de contato e inscrição
└── img/               # Imagens do projeto
```

## :bookmark: Seções

- **Home** — Apresentação da escola
- **Cursos** — Informações sobre os cursos técnicos oferecidos
- **Gestão** — Equipe Gestora
- **Contato** — Formulário para contato e prestação de provas

## :rocket: Como executar

1. Clone o repositório
2. Inicie um servidor PHP local:

```bash
php -S localhost:8000
```

3. Acesse `http://localhost:8000` no navegador

> **Nota:** Necessário ter o PHP instalado. O projeto foi desenvolvido para ambiente de desenvolvimento local com XAMPP ou similar.

## :bulb: Funcionalidades

- Navegação entre páginas
- Formulário de contato com validação
- Design responsivo
- Interface com ícones Remix Icon
