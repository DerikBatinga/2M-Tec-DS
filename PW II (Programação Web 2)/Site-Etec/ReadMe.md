# Site EtecZL

Website institucional da **Etec Zona Leste**, uma escola técnica vinculada ao Centro Paula Souza (CEETEPS).

## :computer: Tecnologias

- **PHP** — Backend e estruturação das páginas
- **HTML5** — Marcação semântica
- **CSS3** — Estilização (layout, cores, tipografia)
- **JavaScript** — Interatividade

## :file_folder: Estrutura

```
Site-Etec/
├── index.php          # Página inicial
├── index.css          # Estilos da home
├── Cursos/            # Página de cursos técnicos
├── Gestão/            # Página sobre a gestão escolar
├── Contato/           # Formulário de contato e inscrição
└── img/               # Imagens do projeto
```
